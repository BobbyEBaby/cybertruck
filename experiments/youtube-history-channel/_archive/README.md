# _archive/

Preserved work from before the 2026-04-11 channel pivot.

## What's here

- **`01-dancing-plague-1518.md`** — full ~1,342-word episode script written by the remote agent (Opus run #10) under the original "obscure history" channel concept. Includes 4 human-voice-segment markers, inline source citations, and a complete narrative arc. Genuinely good work that just happens to be off-theme now that the channel is "rebellions across history with thematic through-lines."
- **`assets-dancing-plague-1518/sources.md`** — the same remote run's B-017 output: 22 public-domain image candidates mapped to narration beats. Note: this is a *manifest*, not downloaded files — the remote sandbox is network-read-only for arbitrary downloads, so the file was designed as a handoff for a local-run step that never ran.

## Why archived, not deleted

Three reasons:
1. **The work is legitimately good.** The script is carefully researched, properly cited, and well-structured. Deleting it would throw away real hours of agent effort.
2. **The Dancing Plague is still an interesting historical topic.** It just isn't a *rebellion*. If this channel ever spins off a second show (or if Breaking Points ever does a "mass hysteria" themed episode), the research here is reusable without redoing it.
3. **Honest record-keeping.** The runlog already records that this work was done; deleting the actual artifact would make the runlog entry misleading.

## Will anything ever happen with this?

Probably not, and that's fine. The default is: leave it here indefinitely. If at some point Breaking Points decides to cover the Strasbourg revolts of 1515/1525 or any mass-hysteria-adjacent event in that region, this material gives a head start. Otherwise it's a preserved reference point.
